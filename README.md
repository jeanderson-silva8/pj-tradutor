## 🗣️ Sobre o Projeto

O **PJ Tradutor** é uma aplicação web de tradução com interface visualmente impactante. O projeto combina funcionalidade prática com design premium, utilizando backgrounds imersivos e estilização artesanal para elevar uma ferramenta do dia a dia ao patamar de experiência visual.

### ✨ Features

- 🌍 **Tradução Instantânea** — Interface limpa e rápida para tradução de textos
- 🎨 **Design Premium** — Background cinematográfico com overlay e glassmorphism
- 📱 **Responsivo** — Adaptado para mobile, tablet e desktop
- ⚡ **Zero Dependências** — Código 100% vanilla, sem frameworks ou bibliotecas
- 🖼️ **Visual Imersivo** — Imagens de fundo premium que elevam a experiência
- 🔖 **Favicon Customizado** — Ícone SVG personalizado para identidade visual

## 🚀 Como Executar Localmente

### Requisitos
- Qualquer navegador web moderno (Chrome, Firefox, Edge, Opera, Safari, etc.)

### Rodando

Como este projeto é construído apenas com HTML, CSS e JavaScript puros (sem dependências ou compilação), você pode executá-lo de três maneiras simples:

**Opção 1 (Direto no Navegador):**
Basta abrir o arquivo `index.html` diretamente em seu navegador dando um duplo clique sobre ele.

**Opção 2 (Com VS Code Live Server):**
Se estiver utilizando o VS Code, instale a extensão **Live Server**, abra a pasta do projeto e clique em **"Go Live"** no canto inferior direito para rodar o tradutor com recarregamento em tempo real.

**Opção 3 (Servidor Local Rápido):**
Caso tenha o Node.js ou Python instalados, execute na raiz do projeto:

*Com Node.js (npx serve):*
```bash
npx serve .
```

*Com Python:*
```bash
python -m http.server 8000
```
Em seguida, acesse http://localhost:8000 ou a porta correspondente no seu navegador.
